## What Is This Project?

The **agentzero-dev-knowledgebase** is a community-maintained reference knowledgebase for the [Agent Zero](https://github.com/frdel/agent-zero) AI agent framework. It is organized as an Obsidian-compatible markdown vault designed to serve both **operators** (people running Agent Zero) and **developers** (people extending it).

## Purpose

- Provide a structured, navigable reference for all Agent Zero subsystems
- Bridge the gap between source code and user understanding
- Serve as an AI-consumable knowledge base for agents working with Agent Zero
- Track the framework's evolution (currently aligned with v1.3+)
